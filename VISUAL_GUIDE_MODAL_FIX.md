## ?? User Experience Flow

### Add Faculty:
```
1. Click [+ Add New Faculty]
2. Modal appears centered
3. Fill: Name, Email, Password
4. Click [Add Faculty]
5. ? Success alert appears
6. Page reloads with new faculty
```

### Edit Faculty:
```
1. Click [Edit] on any row
2. Modal opens with current data
3. Modify fields as needed
4. Click [Update]
5. ? Success alert appears
6. Page reloads with updated data
```

### Assign Subjects:
```
1. Click [Assign] on any row
2. Modal shows all subjects
3. Current assignments are pre-checked
4. Check/uncheck subjects
5. Click [Save Assignments]
6. ? Success alert appears
7. Page reloads with new assignments
```

### Delete Faculty:
```
1. Click [Delete] on any row
2. Confirmation dialog appears
3. Click [OK] to confirm
4. Server checks for enrollments
5. If safe: ? Delete and reload
6. If unsafe: ? Show error (has enrollments)
```
