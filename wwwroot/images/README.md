# Images Directory

This directory contains static images for the TutorLiveMentor application.

## Required Images:

### 1. rgmlogo-removebg-preview.jpg
- **Purpose**: Main logo displayed on the home page
- **Size**: Recommended 92px width (as specified in CSS)
- **Format**: JPG or PNG
- **Description**: Your institution's logo with transparent background

### 2. image.jpg
- **Purpose**: Background image for the home page
- **Size**: High resolution recommended (1920x1080 or higher)
- **Format**: JPG
- **Description**: Background image that will be covered with a gradient overlay

## How to Add Your Images:

1. Place your logo file here and name it: `rgmlogo-removebg-preview.jpg`
2. Place your background image here and name it: `image.jpg`
